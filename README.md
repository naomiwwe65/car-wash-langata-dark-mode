# Car-wash Langata (Dark Mode)

Expo React Native app for Westpark Car Wash with dark mode.

## Getting Started

- Install dependencies: `npm install` or `yarn`
- Start the app: `npx expo start`

## Project Structure

- `car-wash/`: Expo app (screens, components, assets)
- `App.tsx`: App entry
